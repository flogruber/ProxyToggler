# :link:ProxyToggler

A lightweight Firefox AddOn to switch the proxy setting between `none`, `system` and `manual` usage.

## Getting Started

### :sparkles:Installing

Download from `Mozilla Add-ons`.

### :pencil2:Customize

Customize the AddOn via the Settings - Tab.
